Curve

A response curve: control points interpolated with a monotone cubic Hermite spline (Fritsch–Carlson), evaluated as y = curve:evaluate(x). The spline never overshoots its control points, which makes it safe for remapping noise into ranges like [0, 1] — shaping terrain falloff, biome thresholds, spawn weights, or any other value. 

local falloff = Atlas.Curve.new({
	Vector2.new(0, 0),
	Vector2.new(0.4, 0.1),
	Vector2.new(0.7, 0.8),
	Vector2.new(1, 1),
})
local height = falloff:evaluate(noiseValue * 0.5 + 0.5)
