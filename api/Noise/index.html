Noise

Noise generation. Houses the Perlin, Simplex, and Voronoi sampler classes plus fractal helpers that layer any sampler into fractional Brownian motion. 

local simplex = Atlas.Noise.Simplex.new(42)
local height = Atlas.Noise.fbm2D(simplex, x * 0.01, z * 0.01, 4)

) → number

Layers 2D noise octaves into fractional Brownian motion. The result is renormalized by total amplitude, so a sampler emitting [-1, 1] stays in [-1, 1] regardless of octave count.
