Sampling

Point scattering over a rectangular region. All samplers are deterministic for a given seed and return points as {Vector2} with coordinates in [0, width) × [0, height). 

local trees = Atlas.Sampling.poissonDisc(512, 512, 8, 42)

Sampling.poissonDisc(
width: number,
height: number,
radius: number,--

Minimum distance between any two points.

seed: number?,--

Any number. Defaults to a random seed.

maxAttempts: number?--

Candidates tried per active point. Defaults to 30.

) → {Vector3}

Scatters points with blue-noise spacing using Bridson's Poisson disc algorithm: no two points are closer than radius, and the region is filled until no more fit. Runs in O(n) via a background acceleration grid.
