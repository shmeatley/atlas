Voronoi

Seeded Worley (cellular) noise. Each integer cell owns one jittered feature point; samples measure the Euclidean distance to the nearest features.

Distances are returned raw: F1 typically falls in [0, 1] and only rarely exceeds it, so ridged patterns like f2 - f1 keep their full contrast. 

local voronoi = Atlas.Noise.Voronoi.new(42)
local f1, f2, id = voronoi:cell2D(x * 0.1, z * 0.1)
